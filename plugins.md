# Official Plugins

## Content Management

 - (GitHub-Sync)[https://github.com/AntCMS-org/GitHub-Sync] is a plugin that can be used to automatically sync a GitHub repository to your AntCMS instance utilizing the cronjob. The official AntCMS website uses exactly that and syncs from the (AntCMS-org/Website-Content)[https://github.com/AntCMS-org/Website-Content] repository.

## Misc

 - (Example-Plugin)[https://github.com/AntCMS-org/Example-Plugin] is a simple plugin that developers can use as reference for developing their own AntCMS plugins.

# Unofficial Plugins

Have a plugin that you would like to see here? Create a pull request over on the (Website-Content)[https://github.com/AntCMS-org/Website-Content] repository!
